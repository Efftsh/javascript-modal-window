<h2>📌 Javascript modal window</h2>

A simple modal window project built with HTML, CSS, and JavaScript.
This project demonstrates how to create an interactive modal that can be opened by multiple buttons and closed using different user interactions such as clicking the close button, clicking outside the modal, or pressing the Escape key.

<h2> 🛠 Technologies</h2>
<ul>
  <li>HTML5</li>
  <li>CSS3</li>
  <liJavaScript (Vanilla JS)></li>
  <li>DOM Manipulation</li>
</ul>

<h2>✨ Features</h2>
<ul>
  <li>Open modal window using multiple buttons</li>
  <li>Close modal using the close button</li>
  <li>Close modal by clicking outside the modal (overlay)</li>
  <li>Close modal by pressing the Escape key</li>
  <li>Uses CSS classes to control visibility</li>
  <li>Simple and responsive modal layout</li>
</ul>

<h2>⚙️ The Process</h2>
<ol>
  <li>Structured the page layout using HTML with buttons, modal content, and overlay.</li>
  <li>Styled the modal and overlay using CSS, including positioning, shadows, and background blur effects.</li>
  <li>Implemented functionality with JavaScript DOM manipulation.</li>
  <li>Used classList.add() and classList.remove() to toggle the hidden class for showing and hiding the modal.</li>
  <li>Added event listeners to handle:</li>
<ul>
  <li>Button clicks to open the modal</li>
  <li>Close button click</li>
  <li>Overlay click</li>
  <li>Keyboard Escape key press</li>
</ul>
</ol>
